### Amazon-Web-Scraping-Project
A Python-based Amazon Web Scraping project built using BeautifulSoup, Requests, and Pandas to extract and analyze product data such as titles, prices, ratings, and reviews from Amazon product pages.
This project demonstrates skills in data collection, cleaning, transformation, and exploratory data analysis (EDA) using real-world e-commerce data.
